## OVERVIEW
This repository accepts documentation contributions that improve
accuracy, clarity, and consistency. All submissions are reviewed
before approval.

## ACCEPTED CONTRIBUTIONS

- Correction of factual inaccuracies
- Improvements that enhance clarity and precision
- Additions that fill verified documentation gaps
- Spelling, grammar, and typographical corrections

## NON-ACCEPTED CONTRIBUTIONS

- Whitespace-only or formatting-only changes
- Subjective wording preferences without measurable improvement
- Structural or organizational modifications
- Redundant rewrites of existing content
- Documentation of unreleased or experimental features
- Changes to Community Resources

## DOCUMENTATION STANDARDS

- Language must remain professional and objective
- Changes must be minimal and focused
- Content must align with existing documentation style
- Examples, if provided, must be properly formatted

## REVIEW PROCESS

All submissions are subject to maintainer review.
Non-compliant contributions may be declined.

By submitting a contribution, you acknowledge and agree
to the standards outlined above.
